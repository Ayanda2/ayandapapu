Greenleaf
Click image to see the site

Greanleaf is an Ecommerce store built using React js. In this store you are able to add and remove products from the basket or checkout page, this was achieved through the use of react-context-api which is similar to Redux. In the checkout page users are able to see the subtotal price of their products and this was achieved through the use of the react-currency-formater. Finally this site is hosted by firebase and uses firebase authentication to allow users to create an account and login. Click the image to see the site.

Netwatch
Click image to see the site

Netwatch is built with React js using data from the movie database api (TMDB API). In this site you are able to find top rated movies, comedy, action, horror and romantic movies including top rated series. As you will notice every time you reload the page the banner movie thumbnail will change .This site allows you to see trailers of your favorite movies and series using react-youtube. This allows the site to play trailers of the selected movie or series from youtube. Due to some inconsistencies from the api click on series's like money heist and stranger things to see the full functionality of the site.
